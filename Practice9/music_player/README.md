# Music Player

## Controls
- `P` - play current track
- `S` - stop
- `N` - next track
- `B` - previous track
- `Q` - quit

## Features
- Playlist support
- Current track display
- Playback progress bar
- Automatic next-track playback

## Notes
- Sample `.wav` files are included in `music/sample_tracks/`.
- The program also supports `.mp3` and `.ogg` files if you add your own tracks.

## Run
```bash
python main.py
```
