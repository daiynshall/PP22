Practice 10

Contents:
- racer/racer.py
- snake/snake.py
- paint/paint.py

Run:
    pip install pygame
    python racer/racer.py
    python snake/snake.py
    python paint/paint.py

Racer:
- Left / Right arrows: move
- Collect coins
- Avoid enemy cars

Snake:
- Arrow keys: move
- Avoid walls and your own body
- Score and level are shown at the top

Paint:
- B = brush
- R = rectangle
- C = circle
- E = eraser
- Click color boxes to change color
